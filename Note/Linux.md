### Linux
#### 常用命令
```
sudo apt-get update：更新软件列表，防止下载老版本软件；sudo为了提权，获取最高权限，apt为高级工具包工具，apt-get与获取安装软件包相关；
sudo apt-get upgrade：把本地已安装的软件同软件列表的版本进行对比，如果版本过低则更新；
pwd：显示当前工作目录的完整路径
cd：切换当前工作路径  cd .切换到当前工作目录
ls：显示当前目录下的所有文件信息
touch：创建和修改文件
mkdir：创建目录
cp：复制文件和目录
rm：删除文件和目录
mv：移动或者重命名文件和目录
find：搜索文件和目录
du：计算文件和目录的占用空间

```
